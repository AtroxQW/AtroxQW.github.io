Redis笔记05-RedisKey基本命令

RedisKey基本命令

Redis 是一个开源（BSD许可），内存存储的数据结构服务器，可用作数据库，高速缓存和消息队列代理。它支持字符串、哈希表、列表、集合、有序集合、为徒、hyperloglogs等数据类型。内置了复制、Lua脚本、LRU收回、事务以及不同级别磁盘持久化功能，同时通过Redis Sentinel提供高可用，通过Redis Cluster提供自动分区。

在Redis中无论什么数据类型，在数据库中都是以key-value形式保存，通过进行对Redis-key的操作，来完成对数据库中数据的操作。

下面学习的命令：

  • exists key：判断某个key是否存在
  • del key：删除键值对
  • move key db：将键值对移动到指定数据库
  • expire key second：设置键值对的过期时间
  • type key：查看value的数据类型

关于TTL命令，Redis的key，通过TTL命令会返回key的过期时间，一般有3种：

  1. 当前key没有设置过期时间，所以会返回-1
  2. 当前key有设置过期时间，而且key已经过期，会返回-2
  3. 当前key有设置过期时间，而且key尚未过期，会返回key的正常剩余时间

关于重命名RENAME和RENAMENX

  • RENAME key newkey修改 key 的名称
  • RENAMENX key newkey仅当 newkey 不存在时，将 key 改名为 newkey 。

更多命令学习：https://www.redis.net.cn/order/
