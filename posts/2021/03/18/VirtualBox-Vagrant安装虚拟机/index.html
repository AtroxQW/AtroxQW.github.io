VirtualBox+Vagrant安装虚拟机

¶虚拟机环境搭建

¶一、安装 Linux 虚拟机

  • 下载安装 VirtualBox：https://www.virtualbox.org/，需要开启 CPU 虚拟化。

  • 下载 Vagrant 操作系统官方镜像仓库：https://app.vagrantup.com/boxes/search

  • 下载 Vagrant：https://www.vagrantup.com/downloads.html

¶二、测试 Vagrant 和 VirtualBox

打开CMD命令行，输入vagrant，查看是否出现命令提示。

打开VirtualBox，更改虚拟机文件存储位置，默认为C盘。

¶三、快速安装虚拟机

  1. 打开 windows CMD窗口，运行命令vagrant init centos/7，即可初始化一个CentOS7虚拟机环境。

  2. 打开该目录下查看生成的 VagrantFile 文件。

  3. 输入vagrant up命令来向 VirtualBox 创建虚拟机，失败。

  4. 先将镜像文件下载到本地：https://vagrantcloud.com/centos/boxes/7/versions/2004.01/providers/virtualbox.box，放到目录下。

  5. 执行命令 box 镜像文件下载到本地，然后重复上面的过程。

  6. 虚拟机启动成功，如下图所示：

  7. 连接上虚拟机 vagrant ssh， 测试 Linux 命令。

¶四、VirtualBox 中虚拟机状态

  1. 完成以上工作后，我们查看Virtual，显示有一个虚拟机正在运行。

  2. 在VirtualBox中可以选定虚拟机，正常关机。

  3. 关机以后，可以在VirtuaBox中进行启动，或者在cmd中输入命令vagrant up启动虚拟机。

¶虚拟机网络设置

¶一、默认的网络配置

在VirtualBox中，默认的网络时用地址转换和端口转发的方式，对后续的开发很不方便。

端口转发的方式是windows中的一个端口，对应Linux虚拟机中的一个端口，需要手工设置。我们希望给虚拟机一个固定IP地址，让它能够和windows相互ping通。

¶二、修改Vagrantfile

我们可以在虚拟机中，修改网卡设置，但是这样太麻烦。可以直接通过修改VagrantFile文件来修改虚拟机的网络配置。

我们可以在VagrantFile找到这样一行被注释的文本：

# config.vm.network "private_network", ip: "192.168.33.10

我们把这一段配置的文本打开，而这行文本的IP地址我们通过cmd查询：

在cmd中，输入ipconfig，找到以太网适配器 VirtualBox Host-Only Network：

上面Vagrantfile中该文本的IP地址应改为：192.168.56.x (x可以随意)和cmd中的IP相对应。

在Vagrantfile中打开私有网络配置并修改为：config.vm.network "private_network", ip: "192.168.56.10"

修改完配置后，使用命令vagrant reload重启虚拟机。

¶三、虚拟机和主机互ping

重启并连接虚拟机，使用命令ip addr，发现IP地址已经为上一步设置的 192.168.56.10

接下来测试虚拟机和主机能否互相ping通，我这里的主机IP为 192.168.3.28

在linux终端中输入ping 192.168.3.28，虚拟机能够ping通windows。

在windows命令行中输入ping 192.168.56.10，windows能ping通虚拟机。
